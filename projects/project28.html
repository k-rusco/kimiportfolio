INCLUDO

UI/UX Design (2026)

In Switzerland, only around 3/4% of all private residential buildings possess the required barrier-free measures. This problem ends up being heavily discriminating against people with all disabilities, and still permeates through current architects' design process. Includo is a digital tool that aims to minimize, if not stop this problem, by integrating barrier-free compliance in the design process. This tool takes the shape of a drag&drop checker, with which architects can scan and check if their projects respect the current ragulations in place (namely, the SIA500 and the Behindertengleichstellungsgesetz, BehiG)
